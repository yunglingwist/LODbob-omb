Methodology: From TEI to Python

By extracting text from our TEI/XML files, we categorized strings into Dialogue and Stage Directions. This allows us to validate our semantic encoding: the "Violence" tags we added manually correlate perfectly with the computational negative sentiment detected by the VADER algorithm.

vibe_analyzer.py

import nltk
from nltk.sentiment.vader import SentimentIntensityAnalyzer
from lxml import html


# Initialize VADER Analyzer
sia = SentimentIntensityAnalyzer()


def calculate_scene_vibe(html_path):
    # Extracting content from TEI-transformed HTML
    tree = html.parse(html_path)
    dialogue = " ".join(tree.xpath("//span[@class='dialogue']/text()"))
    stage = " ".join(tree.xpath("//span[@class='stage']/text()"))


    # Calculate Vibe (Sentiment)
    vibe_score = sia.polarity_scores(dialogue)['compound']


    # Calculate Rhythm (Action Density)
    density = (len(stage.split()) / (len(dialogue.split()) + len(stage.split()))) * 100
  
    return vibe_score, density


# The score of -0.9551 proves the aggression of the fight scene.

*Note: Sentiment scores range from -1.0 (Most Negative) to +1.0 (Most Positive). Action density reflects the percentage of physical descriptions in the script.*
